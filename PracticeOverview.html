Practice Overview
version 1.0

 This is the first release of Practice Overview reports in tableau. These reports include the following dashboards:

 Assign Parameters: This is used for assigning 'optimal' and 'acceptable' time (in minutes) spread over 'Order Priority' and 'Visit Class'.

  Dictation time by shift: This is used for understanding the distribution of actual dictation read time for a selected day through donut chart, sites with median read time tabular report, and, selected site's read time distribution through box plot chart and tabular report.

  Practice Overview: This is used for undertsanding the distribution of dictation time by count over 'Visit Class' and 'Order Priority' for all shifts on a selected day. This report includes a tabular report, stacked bar chart and a box plot graph.

  Date Picker: It works in two steps. First, a choice needs to be selected from 'Select Date' dropdown. If 'Yesterday' or 'Today' is selected, reports reflects this date. It defaults to 'Yesterday'. If 'Custom Date' is selected, then, date needs to be picked from 'Custom Date' date picker box.
